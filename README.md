# egg-chicken

A RESTful egg framework based on [FeathersJS](https://feathersjs.com/)

## usage

```bash
$ npm install egg-chicken
```

```js
{
  "name": "awesome-project",
  "egg": {
    "framework": "egg-chicken"
  }
}
```
